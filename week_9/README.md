# IT Helpdesk Ticket Registration System

## Purpose of the Application
A simple command-line ticket registration system for the IT Support team at
City University Malaysia. It allows students to report technical issues
(e.g. login problems, WiFi issues, printer malfunctions, password resets)
and automatically creates a helpdesk ticket with an assigned technician
based on the issue's priority level.

## Tech Stack
- Python 3
- Modular design using three files: `main.py`, `ticket.py`, `display.py`
